# Global Search

Status: completed
Date: 2026-06-09

## Gap

The search box filters only the active topic. A user can search from Profile and miss relevant project, activity, or correction entries.

## Impact

AMM should behave like a memory manager, not a set of isolated category tabs. Search must be a global recall path.

## Acceptance

- Non-empty search queries match entries across memory topics.
- Search result cards show each entry's topic.
- Source and Audit views keep their dedicated behavior.
